# Notebook Directory

This directory contains Jupyter notebooks used during the development, testing, and analysis of the bacterial genome curation pipeline.

## Contents

- **Pipeline testing notebook:**  
  An exploratory notebook used to validate core pipeline functionality, including data extraction, metadata handling, and automated downloading of sequencing data.

- **Analysis and visualization notebook:**  
  A notebook used to generate summary statistics, charts, and figures for interpretation of results and inclusion in reports, posters, and manuscripts.

## Notes

- Notebooks are intended to document the analytical workflow rather than serve as a polished software package.
- Outputs reflect the state of the data and pipeline at the time of analysis.
- Execution order is not strictly enforced, but the testing notebook was developed prior to the visualization notebook.
